Different Kinds of Joins

  • inner_join()
  • left_join()
  • right_join()
  • full_join()
  • anti_join()

Inner Join

  • inner_join()
    • Attaching columns together based on a shared key
      • If observation isn’t shared, it is removed

Outer Joins

  • Attaching columns together based on a shared key
    • If observation isn’t shared but needs to be included, we have the following options:

Outer Joins

  • left_join()
    • Maintains observations from first data frame, not from second if they are not shared
  • right_join()
    • Maintains observations from second data frame, not from first if they are not shared
  • full_join()
    • Maintains observations from both data frames, results in NAs for variables that do not exist for given observation

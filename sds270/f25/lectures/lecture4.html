Ways to subset via []

Let’s practice on some five-element numeric vector, which is:

x <- c(2.4, 5.6, -1.3, 2.2, 4.9)

Ways to subset via []

  1. Positive integers
    • Intuitive way to access, indexing starts at 1
x[1]
[1] 2.4
  1. Negative integers
    • For exclusion (not reverse ordering), returns remaining elements
x[-1]
[1]  5.6 -1.3  2.2  4.9

Ways to subset via []

  1. Logical vectors
    • TRUE at some index results in that element kept
x[c(TRUE, FALSE, FALSE, TRUE, TRUE)]
[1] 2.4 2.2 4.9

Question: What happens when the logical vector is not of the same length as the vector we are subsetting from?

Ways to subset via []

  1. Character vectors
    • Useful if a vector is named
names(x) <- c("a", "b", "c", "d", "e")
x[c("a", "c")]
   a    c 
 2.4 -1.3

Matrices and arrays

  • Use of multiple vectors, single vector, or matrix with higher-dimensional data structures
    • My experience: matrix is typically not used
myMat <- matrix(1:9, nrow = 3, ncol = 3)
myMat[2, 3]
[1] 8
myMat[2:3, c(1, 3)]
     [,1] [,2]
[1,]    2    8
[2,]    3    9

Lists

  • [] returns a list
  • [[]], $ returns element(s) from list
myList <- list("apple", 35:38, TRUE)
myList[2]
[[1]]
[1] 35 36 37 38
myList[[2]]
[1] 35 36 37 38
names(myList) <- c("character", "double vector", "logical")
myList$logical
[1] TRUE

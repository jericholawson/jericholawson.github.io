Example: Finding the mean of each column

Suppose I have the following dataframe:

BostonHousing2 |> 
  select(medv, crim, nox, indus) |>
  head(3)
  medv    crim   nox indus
1 24.0 0.00632 0.538  2.31
2 21.6 0.02731 0.469  7.07
3 34.7 0.02729 0.469  7.07

Our goal is to find the mean of each column. How would we do this by-hand?

Example: Finding the mean of each column by-hand

  1. Look through the medv column
  2. Add up all of the values in the column
  3. Divide by the number of values I have
  4. Write this value down
  5. Repeat for the other columns

Iteration allows us to repeat a process as we traverse across some collection of objects.

Iteration

  • Repeats process as we traverse across some collection of objects
  • The collection?
    • A list or vector
    • Columns/rows in a data frame or matrix
  • The foundation?
    • For-loops
    • Vectorized functions
    • Functionals

Example: Finding the mean of each column by-hand

BHsmall <- BostonHousing2 |> 
  select(medv, crim, nox, indus) |>
  slice(1:5)

BHsmall$medv
  1. Add up all of the values in the column

meanVal <- sum(BHsmall$medv) / length(BHsmall$medv)

The for-loop

  • Coding structure to repeat code across a set of items
  • Need the following:
    • index \(i\) or element from the set
    • Code that can be repeated for each element in set or in reference to item in location \(i\)
    • Object to store items as they’re being generated one iteration at a time
  • Referencing Indices
  • Referencing Items
# initialized object
for (i in 1:length(data)){
  # do process here
  # alter initialized object from before
}
# initialized object
for (item in data){
  # do process here
  # alter initialized object from before
}

Example: Finding the mean of each column by-hand

  1. Look through the medv column

meanVal = rep(-1, ncol(BHsmall))

  meanVal[i] <- sum(BHsmall[, i]) / length(BHsmall[, i])

Wrapup on for-loops

  • For-loops in R are inefficient
    • The language was not designed for for-loops…
    • but it was designed for vectorized operations

Vectorized operations

  • Functions built to complete operations on multiple elements at once
    • e.g. colMeans(), +, sqrt()
    • Helpful when there are many elements you want to run operation on

Functionals

  • R functions that use a function name as one of its inputs
    • Why? Allows for efficiency in iteration
    • Can be used on lists, arrays, and data.frames
    • e.g. map() from purrr, across(), apply()

across()

  • Applies a function across multiple columns in a data frame
  • Takes as arguments: the columns to perform the function across and the function name
  • Example 1
  • Example 2
BHsmall |>
  summarize(across(medv:indus, mean))
   medv     crim    nox indus
1 29.98 0.032468 0.4784 4.162

purrr package

  • Included in tidyverse
  • Package for working with functions and vectors
  • Provides a family of map() functions
  • map() functions allow us to apply a function to each element of a list or vector

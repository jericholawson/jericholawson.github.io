Vector types

  • For an imported data set, this is stored as a data.frame object
    • Composed of a series of vectors that make up the columns
    • Each vector is composed of different R types:
      • double: seen with numeric data (2)
      • character: seen for written data (“2”)
      • logical: T, F, or NA
      • Date: represents a calendar date
      • Other types as well
    • May need to factorize a vector as a factor
      • Used for categorical data

Vector types

  • Can check the R type for an object with typeof()
typeof(mtcars$am)
[1] "double"

Hierarchy

  • A vector must consist of elements of the same type
    • integer, logical, double, character

Setting Dates

  • ymd_hms() will take a date formatted as year, month, day, hour, minute, second and convert it to a date time format
  • Before
  • Code
  • After
prisons |> 
  select(NAME, SOURCEDATE) |> 
  head(3)
               NAME             SOURCEDATE
1 BALDWIN PARK JAIL 2018/05/16 00:00:00+00

3      CONCORD JAIL 2018/05/25 00:00:00+00
library(lubridate)

prisons <- 
  prisons |> 
  mutate(SOURCEDATE = ymd_hms(SOURCEDATE))

2    CLAREMONT JAIL 2018-05-25
3      CONCORD JAIL 2018-05-25

Conditionals

  • case_when() allows us to set values when conditions are met

  • Code

prisons <- 
  prisons |>

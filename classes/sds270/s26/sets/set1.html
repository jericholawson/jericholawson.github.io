Background

The first problem set will contain various problems related to the following:

  • Names and values (Ch. 2, AR)
  • Vectors (Ch. 3, AR)
  • Subsetting (Ch. 4, AR)

Make sure to carefully read and follow directions for each problem.

All problems will center around the macroinvertebrates dataset, which is based on a study conducted by the Alaska Science Center over the last decade. The dataset contains lengths of macroinvertebrates collected from rivers in northwest Alaska using drift and kick nets during summer and fall 2015-2019. See the data documentation at this link.

After getting the dataset from the course website, load in the dataset using the read.csv() command.

For any x that is not an atomic vector, return a message (as a character) that states that x is of the wrong type.

Important(c)

Q4: Gambling

Let’s go gambling! We will create functions that run through two different types of games: one that involves a deck of cards and another that involves a spinning wheel.

This question involves using the sample() function in various ways. Read through the documentation to become familiar with how we would sample from a discrete number of outcomes with some probability.

Important(a)

Create a function named cards that takes in the following arguments:

  • wager: amount of money wagered
  • color: “red” or “black”
  • number: 1, 2, …, 13 (assume aces are 1, jacks are 11, queens are 12, and kings are 13)

You will use sample() functions to return a color and number (assume the probabilities are of equal chance). Return winnings, which will be of a combination of these outcomes minus the wager:

  • If card is “red”, bettor wins three times their wager.
  • If card is a face card, bettor wins half their wager.
  • If card is “black” and a 10, bettor wins half their wager.
  • Anything else results in no winnings.

For example, if a red jack is drawn, then the player wins 3.5x their wager back and we will return what they earned minus the wager they put in. As such, $10 wagered will return $25 in winnings.

Important(b)

Create a function named wheel that takes in the following arguments:

  • wager: amount of money wagered

Return winnings, which will be the net gain of the following spin probabilities:

  • 10% chance that they win five times their wager
  • 20% chance that they win their wager back
  • 70% chance that they win no money back

For example, if $10 is wagered and they win on the 5x spot, they will get a net $40 in return.

Important(c)

Create a function named game that passes arguments without addressing them to the card and wheel games within the function. Use a sample() function with an if-else chain to determine which type of game they play, with the following specifications:

  • 30% chance they play cards
  • 70% chance they play the wheel

Return their net gain after playing the game.

Important(d)

Use the replicate() function to play the game 1000 times, with a wager of $10 each time. Take the cumsum() and plot these results across the number of iterations.

Important(e)

Re-do part (c) by using a vectorized if or switch.

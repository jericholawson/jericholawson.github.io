Functional types

apply()

  • base R family of functionals
    • apply(), lapply(), vapply(), tapply(), replicate()

map()

  • From purrr (which is part of tidyverse)
    • map(), map2(), pmap(), walk()

map()

  • Similar to lapply()
    • Apply function to each element of list

Activity

We’ll use the CO2 data below to use the various map() functions:

library(purrr)
co2_df <- as.data.frame(matrix(co2, nrow = 39, ncol = 12, byrow = T))
rownames(co2_df) <- 1959:1997
colnames(co2_df) <- c("Jan", "Feb", "Mar", "Apr", "May", "Jun",
                   "Jul", "Aug", "Sep", "Oct", "Nov", "Dec")
  1. Using the co2_df dataset, subset the data into two different datasets, with the following specifications:
  • co2_60s: Data from 1960-1969
  • co2_70s: Data from 1970-1979

  1. Generate a summary of every single month (column) from the co2_60s data.

  2. Create a function that takes the mean of the difference of two atomic vectors (as inputs x, y) and returns this mean. Then, use one of the map() functions to apply the created function on the co2_60s and co2_70smonths.

  3. Find the minimum monthly CO2 level across the years in the co2_60s data.

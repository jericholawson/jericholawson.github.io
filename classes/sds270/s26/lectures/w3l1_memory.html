Assignment

=

  • More universal
    • Can be used for parameter calls, assignment
y = c(0, 1)

<-

  • For assignment only
    • Technically a function that assigns right side values to left side
    • Question: How could we change the <- symbol such that we assign left side values to the right?
y <- c(0, 1)

Aside: = vs. ==

=

  • For parameter calls and assignment
y = c(0, 1)

==

  • For comparisons between left-sided and right-sided values
    • Results in value/vector of logicals
y == c(0, 2)
[1]  TRUE FALSE

Syntactic Names for Assignment

  • Objects need to be assigned to names consisting of letters, digits, . and _
    • But there are rules:
      • Can’t start with digits or _
      • Can’t use any reserved words (e.g. function, if)
  • General rule: concise, but detailed enough to showcase what it is
    • Can use snakecase, camelcase
# Storing a linear regression model
mod_lr <- lm(mpg ~ disp, data = mtcars)

Copy-on-Modify

  • A copy is only created when modification occurs
    • Atomic vector: copy of whole vector occurs
    • List: copy of elements changed occurs

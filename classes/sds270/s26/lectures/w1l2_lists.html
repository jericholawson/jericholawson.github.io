Lists

  • The “other” vector
  • Flexible with types
    • Technically a collection of objects
  • Can use [[]] to access elements
    • $ can be used if named
    • Question 1: Can we still use []?

List Creation

  • Lists can be created via the list() function
    • vector() function can be used for pre-determined length
      • e.g. vector("list", 4)
obj1 <- "Northampton"
obj2 <- "MA"
obj3 <- c("Abby", "Mike", "Tala")
obj4 <- c("Tunic", "Familiars")

list1 <- list(obj1, obj2, obj3, obj4)
list1
[[1]]
[1] "Northampton"

[[2]]
[1] "MA"

[[3]]
[1] "Abby" "Mike" "Tala"

[[4]]
[1] "Tunic"     "Familiars"

List Creation

  • Can be named within or after creation
    • ____ = ...
    • names()
list(city = obj1, state = obj2, residents = obj3, coffee = obj4)
$city
[1] "Northampton"

$state
[1] "MA"

$residents
[1] "Abby" "Mike" "Tala"

$coffee
[1] "Tunic"     "Familiars"
names(list1) <- c("city", "state", "residents", "coffee")
list1
$city
[1] "Northampton"

$state
[1] "MA"

$residents
[1] "Abby" "Mike" "Tala"

$coffee
[1] "Tunic"     "Familiars"

Concatenation

  • Did you know you can concatenate lists the same way you can atomic vectors??
list2 <- list(interstates = 91)
c(list1, list2)
$city
[1] "Northampton"

$state
[1] "MA"

$residents
[1] "Abby" "Mike" "Tala"

$coffee
[1] "Tunic"     "Familiars"

$interstates
[1] 91

Accessing Elements

list1$city
[1] "Northampton"
list1[[1]]
[1] "Northampton"
list1[1]
$city
[1] "Northampton"

Let’s Add Some Dimension!

  • In most cases, we see data in two dimensions
    • Rows: observations
    • Columns: variables
  • Vectors are inherently one-dimensional
    • Rules and features can add structure to this

Let’s Add Some Dimension!

  • dim attribute adds that dimension
    • If applied to atomic vectors: called a matrix
    • If applied to lists: can be a list-matrix or list-array

Matrices and Arrays

  • Generally for linear-algebra
listMath <- c(2, 4, 3, 2)
dim(listMath) <- c(2, 2)
listMath
     [,1] [,2]
[1,]    2    3
[2,]    4    2
attributes(listMath)
$dim
[1] 2 2

List-Matrices

list1m <- list1
dim(list1m) <- c(2, 2)
list1m
     [,1]          [,2]       
[1,] "Northampton" character,3
[2,] "MA"          character,2

Data Frames

vect1 <- c("SpongeBob", "Patrick", "Sandy")
vect2 <- c("sponge", "starfish", "squirrel")
vect3 <- c(4, 6, 5)

spongeList <- list(character = vect1, species = vect2, height = vect3)
data.frame(spongeList)
  character  species height
1 SpongeBob   sponge      4
2   Patrick starfish      6
3     Sandy squirrel      5

Data Frames

  • A named list of vectors
    • Vectors make up the columns
    • Row entries correspond to collection of individual elements within the vectors

Accessing Elements

  • Done with [[]], with commas in between to denote the elements
    • A little tricky: rows first, then columns
  • To access dimensions while retaining current data structure: [,]
    • Specify indices within dimension you want to subset from
